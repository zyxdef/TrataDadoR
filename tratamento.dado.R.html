Importao dos Dados

Origens possveis para os dados

  • Excel
  • Google Forms
  • Coleta biomtrica de dados (e.g., sensores, actmetro, eye tracking etc.)
  • Dados armazenados em arquivos de outros programas estatsticos (SPSS, Stata, Minitab, SAS etc.)
  • Dados oriundos de sistemas de gerenciamento de bancos de dados (SGBD), extrados via, e.g., SQL
  • Dados no estruturados coletados eletronicamente (e.g., log das buscas realizadas no Google em um intervalo de uma hora)

Crtica dos Dados

Valores No Razoveis

Muitas variveis quantitativas tm intervalos de valores possveis claramente definidos.

Valores fora desses intervalos so suspeitos e devem ser confirmados individualmente.

Exemplo: idade superior a 100 anos

  • no um erro
  • mas raro

Crtica dos Dados

Listagem de Valores a Serem Verificados

Algumas vezes, esses erros podem ser corrigidos na fase de importao e crtica, mas, na maioria da vezes, no.

Por isso, til gerar uma listagem individualizando quais observaes precisam de confirmao (e em quais variveis) e redigitao caso seja constatado erro.

Crtica dos Dados

Listagem de Valores a Serem Verificados

Em algumas vezes, pode ser til que essa listagem seja gerada duas vezes:

  • uma listagem ordenada por observao (til para quem quem for verificar)

  • uma listagem ordenada por varivel (til para quem quem estiver programando)

Integrando Conjuntos de Dados

A integrao de dois ou mais conjuntos de dados envolve um grau de complexidade maior do que a organizao de um conjunto de dados individual.

Integrando Conjuntos de Dados

O grau de complexidade pode variar

  • simplesmente juntar em um conjunto de dados unificado as observaes (linhas) de conjuntos de dados que tenham exatamente as mesmas variveis (colunas)
    • caso mais simples
  • realizar agregaes em nveis diferentes em cada conjunto de dados at que todos os conjunto de dados tenham observaes (linhas) associveis entre si
    • caso bem menos simples
